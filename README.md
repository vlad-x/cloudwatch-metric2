# cloudwatch-metric2

## Description

Get CloudWatch metrics.

## Usage

```
Usage of cloudwatch-metric2:
  -d string
      dimensions
  -delay int
      delay
  -dimensions string
      dimensions
  -f  fail-if-zero
  -fail-if-zero
      fail-if-zero
  -l int
      delay
  -m string
      metric
  -metric string
      metric
  -n string
      namespace
  -namespace string
      namespace
  -p int
      period (default 60)
  -period int
      period (default 60)
  -r string
      region
  -region string
      region
  -s string
      statistics
  -statistics string
      statistics
  -v  version
  -version
      version
```
