# cloudwatch-metric2

## Description

Get CloudWatch metrics.

## Usage

```
Usage of cloudwatch-metric2:
  -delay int
      delay
  -dimensions string
      dimensions
  -fail-if-zero
      fail-if-zero
  -metric string
      metric
  -namespace string
      namespace
  -period int
      period (default 60)
  -region string
      region
  -statistics string
      statistics
  -version
      version
```
